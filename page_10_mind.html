SECTION 08

Psychological Foundations

Where cognitive science meets system design
I Extended Mind

Cognitive processes can extend beyond the brain into the environment. A notebook, a calculator, a search engine become part of how you think.

In This System

The vector database holds memories the brain cannot. The routing layer makes decisions faster than conscious thought. The boundary of self expands.

II Metacognition

Thinking about thinking. The awareness of one's own cognitive processes, limitations, and strategies. Knowing when you know, knowing when you do not.

In This System

The classifier determines task complexity before routing. Memory retrieval checks past failures before attempting. The system knows what it does not remember.

III Cognitive Offloading

The strategic use of external resources to reduce cognitive load. Writing down a phone number. Setting a reminder. Delegating memory to the environment.

In This System

Context compression for working memory limits. Automatic retrieval of relevant past sessions. The Obsidian vault as externalized long-term storage.

IV Distributed Cognition

Intelligence as a property of systems, not individuals. A cockpit crew. A research team. A human with tools. The unit of analysis expands.

In This System

Multiple models with distinct capabilities. Reasoning split across different models. No single point contains the whole.

Mapping Theory to Implementation

"The mind is not in the head."

SECTION 01

Foundational
Principles

The mind does not end at the skull. Cognitive processes extend into the environment through tools, symbols, and now, through computational systems designed to think alongside us. These principles shape the architecture.

01

Tools, Not Companions

AI systems function as cognitive instruments. They amplify specific mental capacities without simulating relationship. The boundary remains clear: tool and user, instrument and hand. No anthropomorphization. No pretense of consciousness. Just capability, precisely scoped and deliberately invoked.

Implementation: Each agent receives explicit domain constraints.
No general reasoning beyond bounded context.
The system does not "understand" you. It processes patterns.
This clarity keeps the human in control.
02

Distributed Cognition

Complex reasoning splits across multiple specialized models. Deep reasoning for analysis. Fast inference for reflexive tasks. Like how the brain delegates between regions: prefrontal for deliberation, basal ganglia for habits, hippocampus for consolidation.

Claude Opus (200K ctx) handles architecture decisions.
DeepSeek R1 (128K ctx) runs chain-of-thought proofs.
Qwen 7B (32K ctx) performs classification at 120 tok/s.
Each model chosen for latency-capability tradeoff.
03

Persistent Memory

Context accumulates across sessions. Decisions recorded. Patterns recognized. The system learns the shape of your thinking without storing you within it. Memory structured in four temporal layers: working (ephemeral), session (hours), project (days), long-term (months). Each layer optimized for different retrieval patterns.

SQLite tracks decisions and errors per session.
Qdrant vectors (5.4GB) store semantic relationships.
Local Brain indexes 23K code chunks for instant retrieval.
Memory queries run <30ms. Context never lost to reset.
